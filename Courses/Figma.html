Curso de Figma

O Design Centrado no Utilizador


O Figma é uma ferramenta essencial para criar interfaces intuitivas e funcionais. Este curso introduz os fundamentos do design centrado no utilizador e ensina como usar o Figma para prototipação rápida.


Criação de Prototipos Interativos


Aprenda a criar protótipos que simulam a experiência real do utilizador. O curso explora componentes reutilizáveis, sistemas de design e colaboração em tempo real.


Exportação e Integração


Com o Figma, é possível preparar designs para desenvolvedores com precisão. Este título aborda como exportar ativos e documentar projetos para facilitar o desenvolvimento.
